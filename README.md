# insilico

Usage of ./insilico:
  -o string
        set the output file. (default "image.png")
  -p float
        set random init percentage. (default 0.5)
  -r int
        set ruleset. (default -1)
  -s int
        set the seed. (default 1583241661879998000)
  -x int
        set the width. (default 640)
  -y int
        set the height. (default 640)
